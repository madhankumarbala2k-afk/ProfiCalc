# 💰 ProfiCalc – Smart Shop Profit Calculator

A simple, offline-first web app to help **small shop owners** manage products, billing, stock, damage, and profit — all from their browser.

## 🚀 Live Demo
👉 [Open ProfiCalc](https://YOUR-USERNAME.github.io/ProfiCalc)

## ✨ Features

- 📊 **Dashboard** — Total Sales, Profit, Damage Loss, Net Profit cards
- ➕ **Add Product** — Name, Stock, Wholesale & Selling Price
- 📦 **Product List** — Search & filter products in real-time
- 🧾 **Billing** — Select product, enter qty, see bill preview, process sale
- ⚠️ **Damage Entry** — Record damaged stock with loss preview
- 📋 **Transactions** — Full history with date, type, and profit/loss
- 🔔 **Low Stock Alerts** — Warning when stock < 5 units
- 💾 **localStorage** — All data saved in browser, no backend needed
